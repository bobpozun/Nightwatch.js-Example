#### Setup Node
* install Node.js
* open Terminal
    * npm install --global generate
    * npm install --global generate-package

#### Setup/run demo test
* install latest version of Java
* clone repo
* cd to repo
* open Terminal
    * npm install
    * npm run {browser} -- --env {environment}
        * example - 'npm run firefox -- --env prod'


#### Update packages and save package.json
* npm update --save
